# 🔬 Welcome to the Ringer UFRJ Project! 👋

Welcome to the official repository hub for the **Ringer UFRJ Group**. We are a research collective from the **Federal University of Rio de Janeiro (UFRJ)**, working in collaboration with **CERN** and the **Signal Processing Laboratory (LPS)**.

Our mission is to develop and maintain high-performance algorithms and frameworks for the **ATLAS Experiment**, specializing in **Fast Egamma Trigger Emulation** and **Machine Learning** applications for high-energy physics. 🌌

---

## 🚀 Our Core Packages

Explore our ecosystem of tools designed for trigger analysis and model development:

| Package | Description | Language |
| :--- | :--- | :---: |
| [**trig-egamma-ntuple**](https://github.com/ringer-ufrj-br/trig-egamma-ntuple) | Extraction of electron and photon ntuples from ATLAS **AOD** format using the Athena framework. 🛠️ | C++ |
| [**trig-egamma-frame**](https://github.com/ringer-ufrj-br/trig-egamma-frame) | A unified Python framework for analyzing trigger e/g ntuples with modular algorithms. 🐍 | Python |
| [**neuralnet**](https://github.com/ringer-ufrj-br/neuralnet) | Research and development of the **Ringer Model** for the **L0 Trigger** in ATLAS. 🧠 | Jupyter |
| [**neuralnet-lightning**](https://github.com/ringer-ufrj-br/neuralnet-lightning) | Scalable implementation of Ringer models using **PyTorch Lightning**. ⚡ | Python |
| [**rootplotlib**](https://github.com/ringer-ufrj-br/rootplotlib) | A plotting library for creating **ATLAS-standard** publications using ROOT and Matplotlib. 📊 | Python |
